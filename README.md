## GitHub Pages 배포 파일 목록

| 파일 | 설명 |
|------|------|
| `index.html` | 메인 앱 (전체 로직·스타일 내장) |
| `manifest.json` | PWA 매니페스트 (아이콘·이름·색상) |
| `sw.js` | Service Worker (오프라인·캐시) |
| `.nojekyll` | GitHub Pages Jekyll 비활성화 |
